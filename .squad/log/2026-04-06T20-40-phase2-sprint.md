## Key Decisions

1. **AD-18 + AD-19 as P0:** Thinking and tool enrichment are small, high-impact changes. Wiring already exists in AgentCore; Gateway just needs to surface it.

2. **Session reconnection protocol:** Separate message types (`reconnect`, `reconnected`, `history_entry`, `history_replay_end`) provide clear semantics.

3. **Isolation options as dictionary:** Keeps contracts extensible without modifying `AgentDescriptor` repeatedly.

4. **Streaming helper extraction (Bender):** Resolves P0 bug (WebSocket streaming history loss) and provides reusable pattern for future streaming scenarios.

5. **Empty stream handling (Hermes decision):** Don't persist empty assistant entries; tool/system entries still recorded.
