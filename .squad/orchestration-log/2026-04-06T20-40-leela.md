## Deliverables

### Architecture Decisions (AD-18 through AD-22)

**AD-18: Thinking Event Pipeline**
- Dedicated `ThinkingDelta` event type in `AgentStreamEventType` enum
- Maps `MessageUpdateEvent.IsThinking` flag to first-class event, not metadata
- WebSocket protocol addition: `{ "type": "thinking_delta", ... }`
- Channel capability matrix: WebUI/TUI support; Discord/Telegram omit

**AD-19: Tool Detail Enrichment**
- Enrich `AgentStreamEvent` with `ToolArguments`, `ToolDurationMs`, `ToolResultTruncated`
- Tool start events carry argument dictionary; tool end events carry duration
- WebSocket protocol enriched for both tool_start and tool_end

**AD-20: Agent Configuration Schema**
- JSON-based agent registration via `IAgentConfigurationSource` interface
- File location: `{AppRoot}/agents/{agent-id}.json`
- Schema includes `systemPromptFile` support, `subAgentIds`, `isolationOptions`
- Code-based registrations take priority over file-based

**AD-21: Isolation Strategy Expansion**
- `IsolationOptions` dictionary on `AgentDescriptor` for strategy-specific config
- Options records: `SandboxIsolationOptions`, `ContainerIsolationOptions`, `RemoteIsolationOptions`
- JSON configuration schema for all three strategies
- Implementation deferred; contracts now ready

**AD-22: Session Reconnection**
- WebSocket `reconnect` message type for session resumption
- History replay protocol with `history_entry` and `history_replay_end` messages
- `ExpiresAt` property on `GatewaySession`; configurable timeout (default 24h)
- Background `SessionCleanupService` for expired session deletion
