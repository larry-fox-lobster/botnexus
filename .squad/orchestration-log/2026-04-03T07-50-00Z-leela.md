## Assignment

Design and implement skills platform — SkillsLoader with global/per-agent discovery, YAML frontmatter parsing, wildcard DisabledSkills filtering, context builder integration, REST API endpoints.

## Deliverables

1. **SkillsLoader** — Dynamic skill discovery and loading from extension folders
   - Global discovery: scan `extensions/skills/` for all available skills
   - Per-agent discovery: filter by agent config + frontmatter directives
   - YAML frontmatter parsing for skill metadata
   - Wildcard support for DisabledSkills filtering (e.g., `disabled-*`)

2. **Context Integration** — Skills injected into agent context at runtime
   - SkillsLoader.GetSkillsForAgent() returns filtered skill list
   - Context builder loads skills into agent prompt

3. **REST API Endpoints**
   - GET /api/skills — list all available skills
   - GET /api/skills/{skillId} — skill details with frontmatter
   - POST /api/agents/{agentId}/skills — bind skill to agent
